# WebAudio Study

WebAudioAPIの練習用リポジトリです。

## ほしい機能
1. 音を再生してみる（一時停止は少しややこしくなるので今回はなし）
2. ミュートの切り替えができる
3. 音源終了時に`audio:ended`イベントが発火する
4. （できれば）フィルターの切り替え、Frequency、Quialityの値を変更できる
5. （できれば）SPで動く
